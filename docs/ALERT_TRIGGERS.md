## 五、冷却规则

| 规则 | 说明 |
|------|------|
| 同股票+同类型+同级别 | 30分钟内不重复发送 |
| 级别升级（warning → critical） | 冷却立即失效，立刻发送 |
| 告警解除 | 从活跃池自动清理 |
| 每封邮件包含 | 本次新告警 + 所有活跃历史告警 |

## 六、当前未触发邮件的情况

以下情况**仅终端显示**，不触发邮件（后续可配置）：

- 自选股普通异动（日线级别信号，如 SMA 金叉/RSI 超买等）
- 成交量温和放大（< 5x）
- 小额涨跌（< 阈值）
- 连接断开/恢复

## 七、触发流程

```
monitor daemon 每轮扫描:
  ① 拉持仓数据 + 退出策略 → 检查止损/止盈/加减仓
  ② 拉自选股快照 → 盘中异动检测
  ③ 拉分钟线快照 → MDP 异常检测
  ④ 汇总告警 → 冷却检查
  ⑤ 需要发送 → 一封邮件包含所有活跃告警
  ⑥ 写入 data/alerts_today.md
```
