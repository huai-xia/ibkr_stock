# PDT 规则（Pattern Day Trader）

> ⚠️ **小资金账户最重要的规则，请务必理解后再交易！**

---

## 什么是 PDT？

**Pattern Day Trader（典型日内交易者）** 是美国金融业监管局（FINRA）制定的一项规则。

如果你的账户总资产 **低于 $25,000**，你在 **5 个连续交易日** 内最多只能进行 **3 次日内交易**。超过 3 次，账户将被标记为 PDT，交易权限会被限制。

---

## 什么是「日内交易」？

**同一天内买入并卖出同一只股票**（或先卖出后买入），算作 **1 次**日内交易。

### ✅ 算作日内交易的例子

```
10:00  买入 AAPL 100 股
14:30  卖出 AAPL 100 股
→ 这算 1 次日内交易
```

```
09:45  卖出 TSLA 50 股（平掉之前的仓位）
10:30  买入 TSLA 50 股
→ 这也算 1 次日内交易（同一天同一股票有买卖）
```

### ❌ 不算日内交易的例子

```
周一  买入 AAPL 100 股
周二  卖出 AAPL 100 股
→ 不算日内交易（跨天）
```

```
10:00  买入 AAPL 100 股
10:01  买入 AAPL 100 股（同一方向，多次买入）
14:00  卖出 AAPL 200 股
→ 算 1 次日内交易（多次买入合并为一次，最终只看当天有无反向操作）
```

---

## 5 日滚动窗口

IBKR 使用**过去 5 个交易日**的滚动窗口来计算日内交易次数，不一定是自然周（周一到周五）。

```
例：
  周一 1 次
  周二 1 次
  周三 0 次
  周四 1 次 → 此时累计 3 次！
  周五 0 次 → 周四那笔仍在窗口内，还是 3 次
  下周一 → 周一的 1 次滚出窗口，剩余 2 次
```

---

## 违规后果

如果 5 日内超过 3 次日内交易：

1. IBKR 将账户标记为 **PDT 账户**
2. 账户被限制交易 **90 天**（只能平仓，不能开新仓）
3. 或者需要在 5 个交易日内将账户资金补足到 **$25,000 以上**

---

## 本系统的 PDT 保护

IBKR 交易助手内置了三层 PDT 保护：

| 次数 | 系统行为 |
|------|----------|
| **0-1 次** | 正常交易，每次日内交易后推送提醒 |
| **2 次** | 🟡 橙色预警。下单前弹出二次确认，微信推送警告 |
| **3 次** | 🔴 硬阻止。拒绝所有日内交易订单 |

**例外**：紧急平仓通道（需要双确认 + 强制记录原因），仅用于突发重大事件。

---

## 如何避免触发 PDT？

1. **尽量持仓过夜**，避免当天买卖同一只股票
2. **收盘前 30 分钟不买新仓位**（以防必须当天卖出）
3. **使用 swing trading 策略**（持仓数天到数周）
4. **时刻关注本系统的 PDT 计数器**：`ibkr-stock pdt`

---

## 补充说明

- PDT 规则是 FINRA 的强制性规定，所有美国券商都必须遵守
- 如果账户资金超过 $25,000，PDT 规则不再适用（可以无限次日内交易）
- 现金账户（Cash Account）不受 PDT 限制，但受 T+2 结算限制（卖出后资金 2 天才能再用）
