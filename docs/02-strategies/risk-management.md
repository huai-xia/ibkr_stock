# 风险管理基础

> 交易的第一要务是**活下去**。盈利其次，风控优先。

---

## 一、为什么风险管理是最重要的？

一个简单的数学事实：

```
亏 10% → 需要赚 11% 才能回本
亏 20% → 需要赚 25%
亏 50% → 需要赚 100%  ← 腰斩后翻倍才能回本
亏 90% → 需要赚 900%
```

**亏损越深，回本越难。** 风控的意义在于：不让自己陷入难以回本的境地。

## 二、仓位管理

### 永远不要 All-in

```
错误：账户 $5000，全部买一只股票
正确：账户 $5000，每只股票最多 $1000（20%）
```

**为什么？** 单票满仓意味着一次意外就全灭。

### 仓位大小的简单公式

```
单笔交易金额 = 账户总资金 × 单票仓位上限（如 20%）
```

### 本系统的仓位保护

| 限制 | 默认值 | 含义 |
|------|--------|------|
| 单票最大仓位 | 20% | 一只股票最多占总资金的 20% |
| 总持仓上限 | 80% | 永远留 20% 现金 |
| 单笔最大亏损 | 2% | 一笔交易最多亏总资金的 2% |

## 三、止损

### 什么是止损？

预先设定一个价格，当股价跌到该价格时自动卖出，限制亏损。

```
买入 AAPL @ $180
止损设在 $175
→ 最多亏 $5/股（约 2.8%）

没有止损：
→ 可能亏 $10、$20、$50...（无下限）
```

### 止损怎么设？

**方法一：技术止损**
- 设在关键支撑位下方
- 突破关键位意味着趋势可能反转

**方法二：百分比止损**
- 买入价的 -2% 到 -5%
- 简单直接，新手推荐

**方法三：波动率止损（ATR）**
- 基于股票的平均波动幅度
- 为高波动股票留更多空间
- 更科学但更复杂

## 四、盈亏比

```
盈亏比 = 期望盈利 / 允许亏损

好的盈亏比 ≥ 2:1
```

**例：**
```
买入 AAPL @ $180
止损 @ $175（亏 $5）
止盈 @ $190（赚 $10）
→ 盈亏比 2:1

即使正确率只有 50%：
10 次交易：5 次对赚 $50，5 次错亏 $25
总收益：+$25
```

## 五、本系统的熔断机制

| 熔断类型 | 触发条件 | 后果 |
|----------|----------|------|
| 日亏损熔断 | 当日累计亏损 ≥ 5% | 停止当日所有交易 |
| 连续亏损熔断 | 连续 3 笔亏损 | 暂停交易 30 分钟 |
| 周亏损熔断 | 当周累计亏损 ≥ 10% | 停止当周所有交易 |

这是**硬限制**，由系统强制执行，不由策略自觉。

## 六、PDT 特别提醒

账户资金 < $25,000 时：
- 5 个交易日内 ≤ 3 次日内交易
- 尽量做 swing trading（持仓过夜）
- **收盘前 30 分钟不买新仓位**

详见 `../03-regulations/pdt-rule.md`

## 七、新手黄金法则

1. **永远设止损**（没有止损不下单）
2. **单笔亏损不超过总资金 2%**
3. **不要越跌越补**（亏损加仓是加速灭亡）
4. **不要因一笔盈利就加大仓位**（过度自信陷阱）
5. **情绪激动时不交易**（愤怒、恐惧、FOMO 都是亏损之源）
6. **每天收盘后回顾交易，记录经验**
