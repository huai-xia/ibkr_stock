## 📖 推荐阅读顺序（新手向）

如果你对股票交易完全陌生，建议按以下顺序阅读：

1. **`01-basics/stock-market.md`** — 理解什么是股票、交易所、市场机制
2. **`01-basics/order-types.md`** — 市价单、限价单、止损单的区别
3. **`03-regulations/pdt-rule.md`** — ⚠️ 小资金账户最重要的规则
4. **`02-strategies/risk-management.md`** — 学会保护本金
5. **`02-strategies/momentum.md`** — 第一个可操作的策略
