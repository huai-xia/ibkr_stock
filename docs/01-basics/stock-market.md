# 股票市场基础

> 股票市场是所有权交易的市场。买入股票 = 成为公司的一部分所有者。

---

## 一、什么是股票？

**股票（Stock/Share）** 代表你对一家公司的所有权份额。

- 持有苹果的 1 股 = 你拥有苹果公司的一小部分
- 股价上涨 → 你的资产增值
- 公司分红 → 你按持股比例获得现金

## 三、交易机制核心概念

### 买卖价差（Bid-Ask Spread）

```
买价（Bid）：买方愿意出的最高价  → $180.50
卖价（Ask）：卖方愿意接受的最低价 → $180.55
价差（Spread）：$0.05
```

- 价差越小，流动性越好（如 AAPL、SPY 价差通常 1 美分）
- 价差是隐性交易成本，对于大成交量影响显著

### 成交量（Volume）

特定时间段内交易的股票数量。

- 高成交量 = 流动性好、买卖容易
- 低成交量 = 买卖困难、价差大
- 避免交易日成交量过低的股票（如 < 10 万股/日）

### 涨跌幅

```
涨跌幅(%) = (当前价 - 前收盘价) / 前收盘价 × 100%
```

美股没有涨跌停板限制（与 A 股不同），但有个股熔断机制（波动过大时暂停交易）。

## 四、交易参与者

| 参与者 | 角色 |
|--------|------|
| **散户（Retail）** | 我们，个人投资者 |
| **机构（Institution）** | 基金、养老金、保险公司（资金量大） |
| **做市商（Market Maker）** | 同时报买卖价，赚取价差，提供流动性 |
| **高频交易（HFT）** | 算法极速交易，利用微小价差套利 |

---

## 五、影响股价的主要因素

### 基本面因素
- 公司业绩（营收、利润、未来预期）
- 行业趋势
- 宏观经济（利率、通胀、就业）
- 地缘政治事件

### 技术面因素
- 供需关系
- 市场情绪
- 技术指标信号

## 六、交易成本

| 成本 | 说明 |
|------|------|
| **佣金** | IBKR 美股通常 $0.0005-$0.0035/股，或固定费率 |
| **价差** | 隐性成本，买入价和卖出价的差 |
| **滑点** | 预期价格和实际成交价的差（市价单常见） |
| **监管费** | SEC 费、TAF 费等，通常极少 |

## 七、关键术语速查

| 术语 | 英文 | 含义 |
|------|------|------|
| 多头 | Long | 买入持有，看涨 |
| 空头 | Short | 借入卖出，看跌 |
| 做多 | Go Long | 买入 |
| 做空 | Short Sell | 借股票卖出，期待下跌后买回还券 |
| 牛市 | Bull Market | 持续上涨的市场 |
| 熊市 | Bear Market | 持续下跌的市场 |
| 支撑位 | Support | 股价下跌时遇到买盘支撑的价格 |
| 阻力位 | Resistance | 股价上涨时遇到卖盘压力的价格 |
| 分红 | Dividend | 公司将利润分配给股东 |
| 市值 | Market Cap | 公司总价值 = 股价 × 总股数 |
